Michael's Palace

An online message board and playground

Bootstrap 3

Michael's Palace is powered by Bootstrap 3. The incredible Mobile First Framework is the best option to run your website.

Node JS

This system utilizes Node JS, along with Express to execute our custom RESTful API service at blistering speeds.

Angular 2

We employ Angular 2, one of the leading JS frameworks to power our front end. The website showcases that web development doesn't have to be complicated.
